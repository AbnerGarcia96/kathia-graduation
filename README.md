# Kathia Graduation Invitation

A React + Firebase invitation page for a graduation celebration.

## Features

- Title section with name, date, and locations
- Countdown timer to the graduation event
- Timeline from high school to college
- Ceremony and party details
- Photo gallery section
- RSVP form with Firebase submission

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Add your Firebase config in `src/firebase.js`.
3. Run locally:
   ```bash
   npm run dev
   ```
